# Undetectable Dex++

This project is an adaptation of the original **Dex++** script created by its authors.  
I do not claim full ownership or authorship — all credit belongs to the original creators.  
My contribution focuses on improving safety and reducing detection, since in some games the unmodified version of Dex++ may trigger kicks or restrictions.

---

## Purpose
- Keep Dex++ functional and useful.
- Improve stability and compatibility across different environments.
- Provide a safer version that respects the work of the original developers.

## Credits

- This repository only provides an adapted version to reduce detection issues.
